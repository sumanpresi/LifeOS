One-time setup

This copy of LifeOS isn't linked to a Supabase project yet. It takes about ten minutes, once, and everything is free:

  1. Create a free project at supabase.com.
  2. In the project's SQL Editor, run the contents of supabase-setup.sql (shipped alongside this file).
  3. Create a GitHub OAuth app at github.com/settings/developers. Set the callback URL to https://YOUR-PROJECT-REF.supabase.co/auth/v1/callback.
  4. In Supabase → Authentication → Providers → GitHub, paste the OAuth app's Client ID and Secret.
  5. In Supabase → Authentication → URL Configuration, set your site's address (e.g. your GitHub Pages URL) as the Site URL.
  6. Open js/config.js in a text editor and fill in SUPABASE_URL and SUPABASE_ANON_KEY.

Note: GitHub sign-in only works from a hosted address (GitHub Pages, Vercel, or a local server) — not from a file opened directly off the disk. Until then, LifeOS keeps working in local-only mode.
